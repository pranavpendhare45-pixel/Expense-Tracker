 Expense Tracker

 📌 Description:-
A simple Expense Tracker web app built using HTML, CSS, and JavaScript. It allows users to add and manage expenses.

 🚀 Features:-
*Add expense transactions
*Display list of transactions
*Calculate  expense totals
*Delete transactions
*Save data using localStorage

🛠️ Tech Stack:-
*HTML
*CSS
*JavaScript

 📂 How to Run:-
1.Download or clone the repository
2.Open `index.html` in your browser
